README

Paste King - это минималистичное пространство для личных текстовых заметок. Авторизуйтесь, создавайте посты, быстро копируйте нужные фрагменты и возвращайтесь к своим записям. Ваши посты надёжно защищены и доступны только после входа в аккаунт, интерфейс управления скрыт до успешной авторизации.
